== Information and neccessary stufss to run the application

* Ruby version '2.2.0'

* System dependencies 'bundle install'


project-famousmusers-repo

Run 'make m' to migrate, reset, and seed the database

Run 'make s' to run server on localhost:3000/

Sign up as a new user or use a testuser from seed.rb

Example:
for normal user(non-moderator) use => username: fake1@fake1.com password: password
for admin user (moderator) use =>     username: admin@admin.com password: password
